# 天地劫·像素传

一款纯前端制作的像素风格回合制战棋游戏，致敬经典国产RPG《天地劫》系列。

## 📖 游戏简介

《天地劫·像素传》是一款完全使用 HTML5 + CSS + JavaScript 制作的静态网页游戏。

- **剧情**：讲述西域少年夏侯仪与千年剑使冰璃跨越千年的宿命之旅
- **玩法**：经典回合制战棋，移动 → 攻击/法术 → 待机
- **章节**：共6章，从河州镇少年到幽城决战
- **角色**：夏侯仪、冰璃、封铃笙、慕容璇玑、古伦德
- **成长**：击败敌人获得经验值，角色升级提升属性

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 选择角色、移动、选择攻击目标 |
| 行动菜单 | 攻击 / 法术 / 道具 / 待机 |
| 结束回合 | 提前结束我方回合 |

## ⚔️ 属性相克

- 火 → 冰 → 雷 → 火
- 暗 ↔ 圣

## 🗺️ 地形效果

| 地形 | 效果 |
|------|------|
| 平地 | 无特殊效果 |
| 树林 | 回避 +20% |
| 山地 | 防御 +2 |
| 幽垠 | 移动力消耗 +1 |

## 🛠️ 技术栈

- 纯前端静态技术（零后端依赖）
- HTML5 Canvas 像素渲染引擎
- Web Audio API 8-bit 音效
- GitHub Pages 免费部署

## 📦 本地运行

```bash

git clone https://github.com/YOUR_USERNAME/REPO_NAME.git
```

## 📝 开发说明

- 所有像素美术资源均由代码生成（16x16 精灵）
- 战棋引擎使用 BFS 计算移动范围
- AI 采用贪心策略：优先攻击，其次靠近
- 关卡、角色、技能数据均在 `js/data.js` 中配置

## 🏆 通关攻略

1. **善用地形**：将角色移动到树林或山地获得加成
2. **保持距离**：法师角色（夏侯仪、慕容璇玑）应保持距离输出
3. **集中火力**：优先消灭敌方法师和残血单位
4. **保护治疗**：封铃笙的治疗法术是续航关键
